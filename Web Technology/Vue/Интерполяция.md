### Текстовая интерполяция


Чтобы динамически вставлять данные (во Vue это называются `связыванием`) в шаблоны используется `mustache` синтаксис (пара фигурных скобок). Значение переменной берется из `data`.

```html
<p>{{ example }}</p>
```

### Вставка HTML

С помощью синтаксиса выше `HTML-код` вставить не получится. Для этих целей есть директива [[Директивы|v-html]]

```html
<p v-html="rawHTML"></p>
```

### Интерполяция в атрибутах

С помощью `mustache` нельзя вставить значение в атрибуты. Здесь поможет директива [[Директивы|v-bind]]. Если же динамическое значение будет `null` или `undefined`, то атрибут удалится.

```html
<p v-bind:data-example="example">Example</p>
```

У HTML элементов атрибуты не могу содержать значения `true` или `false`, поэтому атрибут не исчезает, если динамическое значение `truthy` или `''`, если же значение `falthy`, то атрибут удаляется

```html
<button :disabled="isButtonDisabled">Button</button>
```

### Выражения и функции

Выражение могут использоваться:
1. В текстовой интерполяции;
2. В значениях атрибута в директивах. 

```html
<p>1 + 1 = {{ 1 + 1 }}</p>
<button v-for="n in 5">{{ n }}</button>
```

Вместо выражения может вызываться функция. 

> Такая функция будет вызываться каждый раз при обновлении компонента. Поэтому в ней не должно быть никаких `сайд-эффектов`, например, выполнение асинхронных операций или изменение данных.

```html
<p>Math.random()</p>
```
